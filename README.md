This project also taught me how APIs are handled and decoded to provide the finishe product.

From this course I hope to learn how to build dynamic web pages that do more than just have nice UI components. This would be an added skill for me in the future to use for personal projects such as my own portfolio. I’ve been wanting to create my own portfolio as a web page and have started the process but would love to implement some fun things learned here. I’ve also been wanting to design websites that can handle data input, networks requests such as API calls and more. These will be great skills to have now and for my internship this summer.
